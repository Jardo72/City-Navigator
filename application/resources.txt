# MySQL/MariaDB & Docker
https://hub.docker.com/_/mariadb
https://hub.docker.com/_/mysql/
- MARIADB_ROOT_PASSWORD
- MARIADB_DATABASE
- MARIADB_USER/MARIADB_PASSWORD
- /docker-entrypoint-initdb.d (execute files with extensions .sh, .sql)
- /var/lib/mysql (where MariaDB by default will write its data files)
- default port = 3306
- docker run --detach --name mariadb --env MARIADB_DATABASE=city-navigator --env MARIADB_USER=jardo --env MARIADB_PASSWORD=heslo --env MARIADB_ROOT_PASSWORD=my-secret-pw -p 3306:3306 mariadb:10.10
- docker run --detach --name mysql --env MYSQL_DATABASE=city-navigator --env MYSQL_USER=jardo --env MYSQL_PASSWORD=heslo --env MYSQL_ROOT_PASSWORD=my-secret-pw -p 3306:3306 mysql:5.7.42

- https://dbarrerap.github.io/getting-started-with-mariadb-using-docker-python-and-flask.html
